MACOS APP LAUNCHER / INSTALL GUIDE

VoiceOS

Control your Mac apps with voice. Open, focus, quit, set volume — all through natural language.

STEP 01 — CLONE
Get the source
git clone https://github.com/gabepz/macos-app-launcher.git cd macos-app-launcher
STEP 02 — INSTALL
Install dependencies
npm install chmod +x start.sh
REQUIRES NODE.JS 18+
STEP 03 — CONFIGURE
Add to VoiceOS

Open VoiceOS settings and add this MCP server config. Replace the path with your actual install location.

VOICEOS MCP CONFIG
{ "mcpServers": { "macos-app-launcher": { "command": "/absolute/path/to/macos-app-launcher/start.sh" } } }
STEP 04 — USE
Start talking

Restart VoiceOS. Then just say what you want.

"Open Chrome" "Switch to Slack" "Quit Spotify" "Set volume to 40" "What apps are running?"
AVAILABLE TOOLS
open_app Launch any app by name or alias
focus_app Bring a running app to front
quit_app Gracefully close an app
list_running_apps Show all visible apps
set_volume System volume 0–100
open_url Open any URL in browser
